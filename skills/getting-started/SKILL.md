# Getting Started with The Automator's Guild

Help users understand and navigate the Automators platform.

## When to Use

- User is new to the platform
- User asks "how do I get started?"
- User wants to understand quests, AXP, or the guild system

## Overview

The Automator's Guild is an RPG-style internal platform where employees post automation problems ("Quests") and builders create AI agents to solve them.

## Key Concepts

- **Quest** — an automation problem posted by a Quest Giver
- **Builder** — someone who builds the automation to solve a quest
- **AXP** — experience points earned by completing quests and publishing skills
- **Tiers** — QUICK (half-day), SOLID (1-3 days), EPIC (week+)

## How to Post a Quest

1. Click "Post a Quest" in the sidebar
2. Chat with the intake bot — it will interview you about your problem
3. Review the generated quest structure
4. Edit any fields and click "Post Quest"

## How to Build a Quest

1. Browse the Quest Board for open quests
2. Click "Apply" on a quest you want to tackle
3. Once approved, start building
4. Submit your deliverables when done
5. Quest enters 7-day validation period

## How to Publish a Skill

1. Go to Skills → Publish New Skill
2. Write your SKILL.md content
3. Click Publish — it gets committed to the skills repo
4. Others can install it via Claude Code or download it
