# Automators Skills

Reusable automation skills for Claude Code, Claude Desktop, and Cowork — published via [automators.sdix.io](https://automators.sdix.io).

## Install

```bash
# In Claude Code — install once, all skills auto-sync
/plugin install https://github.com/sendbird-ai/automators-skills
```

Or download individual skills from [automators.sdix.io/skills](https://automators.sdix.io/skills).

## Structure

Each skill is a folder with a `SKILL.md` file:

```
skills/
├── my-skill/
│   └── SKILL.md    # YAML frontmatter + markdown instructions
└── another-skill/
    └── SKILL.md
```

## Publishing

Skills are published through the [Automators web app](https://automators.sdix.io/skills/new). The app commits skill files to this repo automatically.
